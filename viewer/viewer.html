    Quick FAQ

    • Empty poster name or a poster ID identical to the group ID means the message was posted by the group admin anonymously (Telegram’s “hide name” / post-as-group feature).
    • Even when Poster Name is empty, the Poster ID is still recorded and can be used to track or revisit the sender. For anonymous admin posts it will be the same as the Group ID.
    • To visit a user or group page from a recorded message, click the Poster ID in the table or open https://web.telegram.org/k/# in a new tab, replacing <Poster ID> with the ID shown.
    • Curly braces such as {} or {party-popper} in message content mark a custom sticker or emoji that does not map to a standard Unicode emoji. Empty {} means no description was available.
    • Media links shown in the Media column (e.g. blob:... or stream:...) are Telegram runtime URLs. They may no longer be accessible after the Telegram tab is closed or refreshed, even if the link is still displayed. Only media files actually downloaded to the group’s media/ folder can be reopened reliably.
    Click “Open Folder” and select a single group folder
    (e.g. ~/Downloads/telegram-recorder/-5491281397).
